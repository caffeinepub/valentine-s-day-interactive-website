# Valentine's Day Interactive Website

## Overview
A playful Valentine's Day website that asks "Will you be my Valentine?" with interactive buttons and romantic animations.

## Core Features

### Main Question Page
- Display the question "Will you be my Valentine?" prominently
- Two buttons: "Yes" and "No"
- Enhanced "No" button behavior: moves unpredictably in random directions when cursor gets close, always staying visible but evasive
- "Yes" button shows heading "I knew you would say yes. I love you baby." immediately when clicked, then transitions to envelope page
- Smooth button animations and transitions
- Romantic color palette (soft pink/red theme)

### Envelope Interaction Page
- Display a centered envelope image
- Interactive envelope that opens when clicked with smooth CSS/keyframe animation
- Opening triggers release of exactly 10 random romantic messages that spread dynamically across the full screen
- Messages are positioned to ensure each remains clearly visible without overlapping
- Messages use simple floating animations and remain visible and readable throughout
- Enhanced floating heart effects burst in background when envelope opens and continue throughout
- Each envelope opening selects random messages from expanded collection

### Celebration Card Display
- After envelope opens, display an enlarged main card with plain background
- Card features completely plain background with single cute heart graphic centered on top
- No patterns or decorative textures on card background
- Card displays the text: "I knew you would say yes. I love you baby."
- Card design maintains romantic, soft aesthetic while being clean and simple
- Larger card size for better visual impact

### Visual Design
- Cute, lighthearted romantic aesthetic
- Heart decorations and animations
- Smooth transitions between all interactions
- Cheerful, romantic typography
- Responsive design for different screen sizes
- Romantic, soft, and sweet overall tone
- Continuous floating heart animations in background

## Technical Requirements

### Frontend-Only Application
- All functionality runs in the frontend
- No backend data storage required
- Expanded preset list of at least 20 romantic messages stored in frontend code
- Random message selection handled client-side
- Envelope opening animation and message floating effects with dynamic full-screen positioning
- Messages positioned algorithmically to avoid overlap and ensure visibility
- Enhanced floating heart burst animations that continue throughout celebration
- Smooth transition from "Yes" button click to celebration page

### Interactions
- Mouse proximity detection for "No" button evasive movement
- Click handlers for button interactions and envelope opening
- Immediate heading display on "Yes" button click
- Page transitions between question and envelope views
- Dynamic message positioning across full screen
- Enhanced floating heart burst animations
- Responsive button behaviors
- Harmonious integration of envelope animation, floating hearts, and message display

## Content
- Language: English
- Expanded romantic messages collection (at least 20 sweet, loving phrases)
- Valentine's Day themed text and styling
- Main heading and card text: "I knew you would say yes. I love you baby."
